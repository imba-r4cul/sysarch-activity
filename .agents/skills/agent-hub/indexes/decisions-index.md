Records **architecture and scope decisions** for the agent-hub package. Each entry should map to a filled note under `docs/decisions/` using `templates/architecture-note-template.md`.

## When to add a row

- Changing what is in scope for `agent-hub/` versus host-only or optional subtrees.
- Declaring a new canonical command or deprecating a workflow.
- Splitting or merging architecture layers in a way that affects reading order.
